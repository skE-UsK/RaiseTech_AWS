## GitHubの流れ
1. アクセストークンの作成
2. リポジトリの作成
3. クローンと移動
```sh
git clone URL
```
4. ブランチの作成と切替
```sh
git git switch -c test
```
5. ファイルの作成・変更
6. ファイルのステージング
```sh
git add -A
```
7. ファイルのコミット
```sh
git commit -m "コメント"
```
8. ステージング情報のプッシュ
```sh
git push origin branchName
```
9. プルリクエスト
10. マージ
11. ローカルリポジトリの更新
